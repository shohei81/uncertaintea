# UncertainTea

UncertainTea is an experimental probabilistic programming project in Julia.
The goal is a GPU-native inference stack with a Gen-like frontend and a static,
GPU-friendly execution model underneath.

At the current stage, the repository is documentation-first. The immediate goal is
to lock down the architecture, the surface syntax, and the static subset that can
realistically target GPU execution.
